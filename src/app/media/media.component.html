Identity Guidelines

The Logo

The Gazebo logo is a combination of the "GAZEBO" wordmark and the fire symbol. The logo should always appear with these two elements positioned together as shown in one of these two examples.

Color variations

  1. Color

    This is the preferred color version of the Gazebo logo.

  2. Black

    When it is not possible to print using color, the logo should be reproduced in one solid color. When reproducing the logo on black and white printers, or in black ink, use this solid color version. Do not create a gray variation by screening or using a tint of black.

  3. Negative / White

    A negative variation of the Gazebo logo can be "reversed" from black using a solid white version.

Typography

The type family chosen for the Gazebo identity is Montserrat, which can be acquired from Google's font library.
